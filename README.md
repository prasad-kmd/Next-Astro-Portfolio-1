# Premium Dual-Framework Portfolio

This repository contains two visually identical, high-performance portfolio implementations using **Next.js** and **Astro**.

## Project Overview

Built with a premium, contemporary aesthetic, this portfolio features rich interactive components, layered glassmorphism, and subtle animations. It demonstrates modern web development practices across two leading frameworks while maintaining a consistent design system.

### Key Features
- **Shared Design Language**: Consistent dark theme with deep charcoal, noise textures, and electric accents.
- **Sticky Floating Navbar**: Pill-shaped navigation with backdrop blur.
- **Collapsible Sidebar**: Slim left-edge sidebar for social links and quick navigation.
- **Custom Context Menu**: Custom right-click menu overriding the default browser menu.
- **Responsive Multi-Column Layouts**: Optimized for mobile, tablet, and desktop.
- **Micro-interactions**: Polished hover states, scale transforms, and smooth transitions.

## Tech Stack
- **Next.js 15+** (App Router)
- **Astro 6+** (Islands Architecture)
- **Tailwind CSS 4** (Styling)
- **Radix UI** (Primitives)
- **Framer Motion** (Animations)
- **Lucide React & React Icons** (Iconography)

## Getting Started

### Next.js Version
1. Navigate to the directory:
   ```bash
   cd NextJS
   ```
2. Install dependencies:
   ```bash
   pnpm install
   ```
3. Run the development server:
   ```bash
   pnpm run dev
   ```
4. Build for production:
   ```bash
   pnpm run build
   ```

### Astro Version
1. Navigate to the directory:
   ```bash
   cd Astro
   ```
2. Install dependencies:
   ```bash
   pnpm install
   ```
3. Run the development server:
   ```bash
   pnpm run dev
   ```
4. Build for production:
   ```bash
   pnpm run build
   ```
