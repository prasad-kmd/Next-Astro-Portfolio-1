# Professional Portfolio Website

A high-performance, premium portfolio website implemented in two frameworks: **Next.js** and **Astro**. Both implementations share the exact same design language, components, and behavior.

## 🚀 Projects

- [Next.js Implementation](./NextJS) - Built with Next.js 15, App Router, and Tailwind CSS 4.
- [Astro Implementation](./Astro) - Built with Astro 5, React Islands, and Tailwind CSS 4.

## 🎨 Design Features

- **Custom Theme**: Electric Violet accent on a Deep Navy base.
- **Glassmorphism**: Layered backgrounds and frosted glass effects.
- **Micro-interactions**: Purposful animations using Framer Motion.
- **Responsive**: Fully optimized for mobile, tablet, and desktop.
- **Interactive**:
  - Pill-shaped sticky floating navbar.
  - Collapsible sidebar with mobile drawer behavior.
  - Custom right-click context menu.
  - Animated particle background.

## 🛠️ Tech Stack

- **Frameworks**: Next.js 15, Astro 5
- **Styling**: Tailwind CSS 4, shadcn/ui
- **Primitives**: Radix UI
- **Icons**: Lucide React, React Icons
- **Animations**: Framer Motion

## 📂 Structure

- `Home`: Landing page with hero, skills, projects, and stats.
- `Personal Details`: Structured resume-style profile.
- `Contact`: Functional contact form and social links.
- `About This Site`: Meta-information about the tech stack.

## 🛠️ Development

### Next.js
```bash
cd NextJS
npm install
npm run dev
```

### Astro
```bash
cd Astro
npm install
npm run dev
```
